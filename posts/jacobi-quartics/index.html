Jacobi Quartics

This is a short note on Jacobi quartics and its relationship with elliptic curves.

Definition

An elliptic curve in Jacobi quartic form is a curve of the form $J_a: y^2 = x^4 + 2ax^2 + 1$, where $\det(J)$, which is some multiple of $a^2 - 1$ here, is nonzero. This model of elliptic curves is important as it provides quicker point arithmetic. We shall prove that the curve $J_a$ is birationally equivalent to $E_a: 2v^2 = u^3 - 2au^2 + (a^2 - 1)u$. The origin of $J_a$ is $(0, 1)$.

Jacobi quartic form -> Elliptic Curve

We shall verify that $\varphi: J_a \to E_a$ with $\varphi(x, y) \coloneqq \left(ax^3 + xy + x : ax^2 + y + 1 : x^3\right)$ is indeed a map between curves. We begin by observing that the origins indeed map to each other. For the rest, we set up the coordinate ring of $J_a$ in Sage:

R.<a, x, y> = QQ[]
I = R.ideal(y^2 - (x^4 + 2 * a * x^2 + 1))
J = R.quotient_ring(I).fraction_field()
a, x, y = J.gens()
J
# Fraction Field of Quotient of Multivariate Polynomial Ring in a, x, y over Rational Field by the ideal (-x^4 - 2*a*x^2 + y^2 - 1)

We then verify that the claimed formula actually maps to a point on $E_a$. We use the homogenised equation $v^2w = u^3 - 4au^2w + 4(a^2 - 1)uw^2$ instead:

u, v, w = 2 * (a * x^3 + x * y + x), 4 * (a * x^2 + y + 1), x^3
f = v^2 * w - (u^3 - 4 * a * u^2 * w + 4 * (a^2 - 1) * u * w^2)
assert f == 0

Since the two curves are birationally equivalent, there is also an inverse map. It is not the easiest to write down, but just note that Sage already provides such functionality. For a fixed $a$, we can use the following code:

R.<x, y> = QQ[]
a = 7

C = Curve(y^2 - (x^4 + 2 * a * x^2 + 1))
phi = Jacobian(C, morphism=True)
E = phi.codomain()
Es = EllipticCurve(QQ, [0, -4 * a, 0, 4 * (a^2 - 1), 0])
iso = E.isomorphism_to(Es)
f = iso.as_morphism() * phi

f
