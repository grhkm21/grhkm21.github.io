MA259 Notes

These are some summarised notes for MA259. I tried my best :)

Just like any other module, this module can be split into theoretical and computational. Obviously the theoretical parts are important as well, but in my opinion having a computational intuition helps see the big picture easier. Here, I only summarise the computational part of the module. For the proofs, you will need to look at the lecture notes themselves, but I don’t think they will be terribly important.

Paper summary

The exam will be 2 hours. It consists of 3 questions totalling to 40 + 20 + 20 = 80 marks.

Summary

Below, $p$ is a prime. There are few big parts to the module:

  1. Numbers (Chapter 1.1 - 1.3)

    • Division & Divisibility
    • Primes
      • Fundamental theorem of arithmetic (FTA)
    • Computing GCD / LCM
      • Euclidean algorithm
      • Extended Euclidean algorithm
      • Bezout’s lemma

  2. Basic congruences (Chapter 1.4 - 1.5)

    • Addition, multiplication, “division”
    • Chinese remainder theorem (CRT)
      • Numeric (with modulos)
      • Algebraic (with rings)
    • Linear congruence(s)
      • Single: $ax \equiv b \pmod{m}$
      • System: $x \equiv a_i \pmod {b_i}$
    • Descent
      • Infinite descent

  3. More congruences (Chapter 1.5 - 1.9)

    • Euler totient function
      • Definition: $\varphi(n) = \left|\left(\mathbb{Z} / n\mathbb{Z}\right)^{\times}\right|$
      • Identities: $\varphi(pq) = \varphi(p)\varphi(q)$ and $\sum_{d \mid n} \varphi(d) = n$
    • Euler’s theorem & Fermat’s little theorem
      • Simplify: $a^{\varphi(n)} \pmod{n}$, $a^{p - 1} \pmod{p}$, $13^{200} \pmod {19}$
    • Wilson’s theorem
      • Reduce $(p - 1)! \pmod{p}$
    • Primitive roots
      • Existence, number of them
      • Primitive roots in $\textrm{mod} \,\, p^k$
    • Quadratic residues
      • Legendre symbol $\left(\frac{a}{p}\right)$
      • Euler’s criterion $\left(\frac{a}{p}\right) \equiv a^{(p - 1) / 2} \pmod p$
      • Quadratic reciprocity $\left(\frac{p}{q}\right)\left(\frac{q}{p}\right) = (-1)^{\frac{p - 1}{2}\frac{q - 1}{2}}$
      • And its supplementary laws for $\left(\frac{-1}{p}\right)$ and $\left(\frac{2}{p}\right)$

  4. Lattices (Chapter 2.1 - 2.2)

    • Minkowski’s theorem
      • Statement
      • Proving Two-square theorem, four-square theorem

  5. Gaussian integers (Chapter 2.3)

    • Gaussian integers
      • Definition: $\mathbb{Z}[i] = a + bi$
      • Norm: $N(a + bi) = a^2 + b^2$
      • Multiplicative norm
      • Bezout’s lemma applies
    • Gaussian primes
      • Classification (up to units): All primes $p \equiv 3 \pmod{4}$ or $\alpha$ where $N(\alpha)$ is prime
      • FTA into Gaussian primes
      • Factorisation algorithm: For $p \mid N(\alpha)$,
        • If $p = 2$ then $1 + i \mid \alpha$
        • If $p \equiv 3 \pmod{4}$ then $p \mid \alpha$
        • Otherwise, $p = a^2 + b^2 = (a + bi)(a - bi)$ are primes

  6. Pythagorean triples (Chapter 2.4 - 2.5)

    • Parametrisation: $(x, y, z) = (u^2 - v^2, 2uv, u^2 + v^2)$
    • Solving $ax^2 + by^2 = cz^2$
      • Criterion: $(bc, ac, -ab)$ is a quadratic residue mod $(a, b, c)$.

  7. $p$-adic numbers (Chapter 2.6 - 2.7)

    • $p$-adic integers
      • Definition: Sequence $(x_1, x_2, \cdots) \in \mathbb{Z}_p$ where $x_{n + 1} \equiv x_n \pmod{p^n}$
      • Termwise addition and multiplication
      • Units in $\mathbb{Z}_p$: All numbers where $x_1 \neq 0$
    • Hensel’s lemma
      • Statement: If $v_p(f’(x)) = k$ and $v_p(f(x)) \geq 2k + 1$, then $f$ has a root $y$ where $y \equiv x \pmod{p^{k + 1}}$
      • Actually computing the roots.

  1. Prove that an integer is divisible by $9$ if and only if the sum of digits is divisible by $9$.

  2. Compute $g = \gcd(1038, 2628)$, and find integers $x, y$ such that $1038x + 2628y = g$. Also parametrise all solutions to the equation.

  3. Find the inverse of $7$ mod $55$. From this, solve $7x \equiv 5 \pmod{55}$. Also solve $35x \equiv 10\pmod{55}$.

  4. How many zeros are at the end of $1000!$ ($1000$ factorial)?

Hint Hint

Lattices (Chapter 2.1 - 2.2)

  1. Show that if $p = a^2 + 2b^2$, then $p \equiv 1$ or $3 \pmod{8}$.

  2. Suppose that $p$ is a prime satisfying $p \equiv 1$ or $3 \pmod{8}$.

    a) Show that $x^2 \equiv -2 \pmod{p}$ has a solution.

    b) Using the solution to $x^2 \equiv -2 \pmod{p}$, construct a lattice $L$ with the property that $(a, b) \in L \implies a^2 + 2b^2 \equiv 0 \pmod{p}$.

    c) Let $S$ be the ellipse $\{(x, y) \in \mathbb{R}^2 : x^2 + 2y^2 < 2p\}$. Show that $S \cap L$ contains a nonzero vector.

    d) Finally, conclude that a prime $p$ can be written in the form $p = a^2 + 2b^2$ if and only if $p \equiv 1$ or $3 \pmod{8}$.

Gaussian integers (Chapter 2.3)

  1. Compute $g = \gcd(14 + 32i, 2 + 16i)$. Hence, find $\alpha, \beta \in \mathbb{Z}[i]$ such that $(14 + 32i)\alpha + (2 + 16i)\beta = g$.

  2. Write $2205$ as a sum of two squares. Hence, factor $2205$ into product of Gaussian primes. Justify that the factors you give are prime.

  3. Find a prime $p$ that is not a Gaussian prime.

  4. Factor the following into product of Gaussian primes: $4 - 3i, 5 + 7i, 1 - i, 2 + 3i, 20 + 17i$.

Assignment 4, question 5.

  2. Recall that $\mathbb{Z}_p^{\times}$ denotes the $p$-adic integers which are units / invertible. Fix $p = 31$ and define the function $f: \mathbb{Z}_p^{\times} \to \mathbb{Z}_p$ where $f(x) = \frac{x^2 + 14}{2x}$.

    c) Define the $p$-adic absolute value $|\cdot|_p$.

    d) Prove that $|f(f(x)) - f(x)| \leq |f(x) - x|^2$ and $|f(x)^2 - 14| \leq |x^2 - 14|^2$.

By considering the sequence $S_2 = (a^2, f(a)^2, f(f(a))^2, \cdots)$, prove that the limit of $S_1$ satisfies $\left(\lim S_1\right)^2 - 14 = 0$. Hence, iterating $f$ gives a method to compute $\sqrt{14}$ in $\mathbb{Z}_p$.

Hint (b) Hint (b)
It suffices to show that $p \nmid x \implies p \nmid f(x)$, which follows from simple algebraic manipulation.
Hint (d) Hint (d)
Expand everything and complete the square.
Hint (e) Hint (e)
First note that $p \mid f(a) - a$, so $|f(a) - a| \leq p^{-1}$, and hence the sequence converges. Also note that $|a^2 - 14| \leq p^{-1}$, so the sequence $(a^2, f(a)^2, \cdots)$ converges to $14$, meaning the limit is just $\sqrt{14}$ in $\mathbb{Z}_p$.
